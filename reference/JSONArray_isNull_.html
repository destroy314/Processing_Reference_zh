Class

JSONArray

名称

isNull()
示例
JSONArray json;

void setup() {

  json = new JSONArray();

  json.setInt(0, 32);
  json.setString(1, null);

  if (json.isNull(1) == true) {
    println("This element is undefined");
  } else {
    println("The value is " + json.getString(1));
  }
}
描述 Determines if the value associated with the index is null, that is has no defined value (false) or if it has a value (true).
语法
.isNull(index)
参数
index int: must be between 0 and length() - 1
返回boolean
